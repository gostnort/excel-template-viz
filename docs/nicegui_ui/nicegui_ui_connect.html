选择授权文件：系统文件对话框选择 OAuth 客户端 JSON → 写入 credentials/oauth_client.json，「连接」变为可用。
连接：首次可能打开浏览器完成 OAuth；成功后 AutoConnect(conn).run(cfg) 加载 TOML 中 id 所在工作表（无 id 则第一个 field_rules 需要的工作表）并刷新下方 HTML5 表。
删除：cancel_auth()，移除授权文件并清空连接。
模板激活后若已存在授权文件，自动调用 core_connect.AutoConnect.run()，无需再点「连接」。
